# ArbiLearn

🚀 **Live Demo:** (https://arbitrum-builder-labs-workshop.vercel.app/)

## Project Description
ArbiLearn is a visual, hands-on website designed to make Ethereum scaling and Layer 2 concepts clear. It allows users to compare core concepts, track live prices, and mine tiny blocks to see immutability in action.

## Pages Overview
The website consists of four main pages:
- **Home (`index.html`)**: The landing page that provides an overview of Arbitrum, explains why Layer 2 matters, and introduces the core features of the platform.
- **Concepts (`concepts.html`)**: An educational section that dives deeper into Layer 1 vs. Layer 2, rollups, and how Arbitrum scales Ethereum.
- **Live Prices (`prices.html`)**: A dashboard that tracks and displays real-time cryptocurrency prices for relevant assets.
- **Block Simulator (`simulator.html`)**: An interactive tool where users can mine tiny blocks and visualize how blockchain transactions, hashing, and immutability work in practice.

## How to Install and Run Locally
Since this is a static website (HTML, CSS, JS), you don't need any complex build tools to run it.

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/Torototooo/Arbitrum_Builder_Labs_Workshop.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Arbitrum_Builder_Labs_Workshop
   ```
3. Open the `index.html` file directly in your preferred web browser. 
   *(Alternatively, you can use a local development server like the "Live Server" extension in VS Code for a better development experience).*
